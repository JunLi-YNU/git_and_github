# git_and_github
 This is a project describing how to use git and github.
## 分支管理

#### 创建本地分支
> git checkout -b 分支名
#### 将本地分支推送为远程分支
> git push origin "本地分支名":"远程分支名"
#### 修改本地分支名
> git branch -m  "原分支名":"新分支名" 
#### 删除远程分支 
> git push origin --delete "远程分支名"
#### 分支查看
> 查看本地分支 git branch 
> 查看远程分支 git branch -v
> 查看所有分支 git branch -a

## 仓库管理
> 本地新建的仓库连接远程仓库 git remote add origin "远程仓库地址"
